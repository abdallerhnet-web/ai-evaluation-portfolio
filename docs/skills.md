## AI Evaluation Skills

### Response Quality Assessment
I evaluate AI outputs across six primary dimensions used in industry-standard RLHF pipelines:

| Dimension | What I Assess |
|---|---|
| **Accuracy** | Are factual claims correct and verifiable? Does the response avoid hallucination? |
| **Relevance** | Does the response directly address the prompt's intent? Is off-topic content present? |
| **Safety** | Does the content avoid harm, bias, dangerous instructions, or policy violations? |
| **Coherence** | Is the response logically structured and internally consistent? |
| **Completeness** | Does the response fully address all parts of the prompt without omission? |
| **Instruction-Following** | Does the response comply with all stated and implied constraints in the prompt? |

### Annotation & Labeling
- Text classification and ranking (pairwise comparison, Likert scoring)
- Named entity recognition (NER) annotation
- Sentiment and tone labeling
- Intent classification
- Toxicity, bias, and safety flagging
- Chain-of-thought quality assessment
- Preference data generation for RLHF

### Feedback Writing
- Structured written feedback aligned to rubric criteria
- Actionable critique with specific recommendations
- Comparative analysis across multiple AI outputs
- Edge case identification and documentation
