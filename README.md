## 🎯 What I Do

### AI Response Evaluation
I assess AI outputs against structured rubrics covering **accuracy**, **relevance**, **safety**, **coherence**, **completeness**, and **instruction-following**. I produce detailed written feedback that directly supports model improvement and RLHF training pipelines.

### Prompt Engineering
I design, test, and iterate prompts across GPT-4, Claude, and Codex — identifying failure modes, boundary conditions, and optimization opportunities. I document results in structured reports suitable for research and training datasets.

### Technical Writing
I produce clear, structured documentation — how-to guides, API references, knowledge base articles, and workflow guides — for AI tools and developer platforms. All writing adheres to professional style guides with version control via GitHub.

### Data Annotation & Fact-Checking
I annotate text, validate factual claims using 3+ independent sources, identify hallucinations, and flag biased or unsafe content in AI outputs. I follow annotation guidelines at the same standard as Scale AI and Surge AI rubrics.
